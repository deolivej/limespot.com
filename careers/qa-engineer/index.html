QA Engineer

Job Description

We are seeking a creative and skilled senior QA Engineer in Test to join the LimeSpot development team. The successful candidate should have a good understanding of Object Oriented concepts and be experienced working as a tester within a Scrum environment. They should be familiar with TDD and Agile testing with a good background in C#, HTML5, CSS3, JavaScript, and AJAX communicating with RESTful web services.

The QA Engineer in Test will work closely with the development team in a Scrum environment in an effort to prevent software defects as opposed to find them. The successful candidate needs to be able to successfully collaborate with developers and participate in release readiness efforts as well as demo development. They also need to be capable of critically reviewing and analyzing user requirements and enforce the Definition of Done within the development team.

Responsibilities

  • Develop a thorough understanding of our technology and the value it provides.
  • Write acceptance criteria for each item for the next sprint.
  • Update acceptance criteria according to PO comments and create test cases for the next sprint.
  • Design and write test cases and unit tests based on backlog items including acceptance criteria for each user story as "Definition of Done".
  • Automate and execute smoke tests UI/Regressions for rapid feedback.
  • Plan and conduct Exploratory Testing, Security Testing and Performance Testing.
