External Contents

External Contents are used to create hyperlinks to external web addresses. Some of the most notable uses of External Contents are:

  • Create hyperlinks to your primary website or online shop.
  • Redirect users to other Social Channels.
  • Value-added links such as "Find Your Nearest Reseller", "Contact Us", etc.
  • Link to blogs and marketing fliers.

External Contents can be accessed from the Products and Contents section of Admin Panel. The landing page for External Contents is shown below:

From this page you can:

  • Create new External Contents
  • Copy existing External Contents
  • Edit External Contents
  • Delete External Contents
  • View External Content as seen by customers during runtime

How to Create External Contents

  1. Select the External Contents tab from the Admin Panel.
  2. Click on the Add External Content button present in the top right corner. You will be presented with following screen:
  3. Enter a Title for the External Content you are creating.
  4. Upload an image that the External Content object will display in your store.
  5. Enter the URL or web address for the webpage your customers will be sent to when they click on the External Content.
  6. Enter a description for internal reference. You customers never see the contents of Description field.
  7. Enter related Targeting Keywords.
  8. Choose appropriate Targeting Rules*.
  9. Click the Save button to complete the process.

* Our system provides a pre-generated list of Targeting Rules that cover most widely used conditions. However, you are free to create custom Targeting Rules from the Admin Panel.

While it is not mandatory to attach targeting rules to External Contents, this feature can be used when you want to associate different contents like image/URL with different user categories. For example, a different marketing image or URL could be used for male and female shoppers. See Add Targeting Rules for more information.

You can display as many External Contents as you want on a single page of your LimeSpot shop. You can click here to learn more about steps of Adding External Contents to Product Catalogs.
