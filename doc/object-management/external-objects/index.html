External Objects

You can use External Contents to create hyperlinks to web address present outside of your LimeSpot powered shop. The external contents can redirect shoppers to your website, primary online store, or any other URL of your choice. You can display as many External Contents as you want on a single page of your LimeSpot shop.

Some of the top uses of External Contents are:

  • Create hyperlinks to primary website or online shop.
  • Redirect users to other Social Channels.
  • Value-added links such as ‘Find Your Nearest Reseller’, ‘Contact Us’ etc.
  • Link to blogs and marketing fliers.

Custom Contents can be accessed from the Admin Panel. The landing page looks like this:

From the landing page of External Contents you can:

  • Create new External Contents
  • Copy existing External Contents
  • Edit External Contents
  • Delete External Contents
  • View External Content as seen by customers during runtime

How to Create External Contents

  1. Select the External Contents tab from Admin Panel.
  2. Click on the + icon present at the top left corner of page. You will be presented with following screen:
  3. Enter a Title for the External Content you are creating.
  4. Enter the URL or web address for the webpage your customers will be sent to when they click on the External Content.
  5. Upload an image that the External Content object will display in your store.
  6. Enter a description for internal reference. You customers never see the contents of Description field.
  7. Choose appropriate Targeting Rules* and click Save.

*An empty list of Targeting Rules indicates that you have not created any Targeting rules. While it is not mandatory to attach targeting rules to External Contents, but this feature can be used when you want to display different content for specific user categories. For example a different marketing image targeted to Male and Female shoppers. See Create and Edit Targeting Rules for more information.
