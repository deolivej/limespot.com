HTML Objects

You can use Custom Contents to create HTML Tags enclosing files of various formats such as multimedia (audio/video), Text, Flash, iFrames, etc. Custom Contents can be added to your online shops to enable several types of HTML supported functionalities.

Some of the top uses of Custom Contents are:

  • To show rich graphics files.
  • To display videos.
  • Create forms for user input.
  • To implement JavaScript.
  • Creative Animations and Visual Effects.

Custom Contents can be accessed from the Admin Panel. The landing page for Custom Contents looks like this:

From the Landing Page of Custom Contents you can:

  • Create new Custom Contents
  • Copy existing Custom Contents
  • Edit Custom Contents
  • Delete Custom Contents
  • View Custom Content as seen by your customers during Runtime

How to Create Custom Contents

  1. Select the Custom Contents tab in the Admin Panel.
  2. Click on the + icon from top left corner of page. You will see the following screen:
  3. Enter the Title for Custom Content you are creating.
  4. Choose an image file that you want to display on your online storefront.
  5. Enter the description for internal reference. You customers never see the contents of Description field.
  6. Enter the HTML code that gets executed when users click on image.
  7. Toggle the Active checkbox on or off, depending on if you want the Custom Content to be visible to your shoppers.
  8. Choose appropriate Targeting Rules.*
  9. Click on the Save button to complete the process.

* Our system provides a pre-generated list of Targeting Rules that cover most widely used conditions. However, you are free to create your own Targeting Rules from the Admin Panel.

Our engine treats Custom Contents just as regular products, and thus you can add targeting rules to them. See Add Targeting Rules for more information.

For example, by adding an appropriate Targeting Rule to Custom Contents, you can show two different marketing videos to men and women. A Targeting Rule would be created to identify the userâ€™s gender and our engine will display the related video accordingly.
